DCAT-AP.EIA Codelisten-Repository / DCAT-AP.EIA Codelist Repository

Dieses Repository enthält die Codelisten, die in der Spezifikation für DCAT-AP.EIA {DCATAPEIA_VERSION} verwendet werden. This repository houses the codelists used in the specification for DCAT-AP.EIA {DCATAPEIA_VERSION} (German).

Die Codelisten können hier abgerufen werden: You can find the available codelists below:

  1. Art eines UVP-Verfahrens / Procedure Type
  2. Dokumententyp / Document Type
  3. Art eines UVP-Verfahrensschrittes / Procedure Step Type
Die DCAT-AP.EIA Spezifikation wird auf github gepflegt: https://github.com/wemove/dcat-ap-eia/ The DCAT-AP.EIA specification is being maintained on github: https://github.com/wemove/dcat-ap-eia/.
